Run a Validator (Kusama)

Running a validator on the Kusama network is identical to running a Polkadot validator. Check out the Polkadot guide on how to setup a validator.

Make sure to adjust the Polkadot guide to run a Kusama network validator:

  1. When starting the node pass --chain=kusama CLI flag:
./target/release/polkadot --pruning=archive --chain kusama
  1. Similar to Polkadot network Kusama has its own token called KSM
